## 2 外键关联规则
1. 用户表中的用户 ID 对应‘成员变更记录’中审核人 ID 和操作人 ID 外键，用来分别标记‘谁发起了变更’和‘谁审批了该变更.
2. 用户表的用户 ID 对应通知表的用户 ID 外键, 用于记录谁发起了通知
3. 用户表的用户 ID 对应通知已读记录表的用户 ID 外键, 用于记录谁读了通知
4. 通知表的通知 ID 对应通知已读记录表通知 ID 外键, 用于记录记录的是哪一条通知
5. 用户表的用户 ID 对应公示表的发布人 ID 外键, 用于记录谁发布了公示.
6. 用户表的用户 ID 对应了公示材料提交记录表的用户 ID 和审核人 ID 外键, 用于记录谁提交了材料和谁审核了材料.
7. 用户表的用户 ID 对应考勤任务表的用户 ID 外键, 用于记录谁创建了考勤.
8. 用户表的用户 ID 对应考勤记录表的用户 ID 和录入人 ID 外键, 用于记录谁录入的考勤情况和录入了谁的考勤情况.
9. 用户表的用户 ID 对应请假申请表的用户 ID 和审核人 ID 外键, 用于记录谁请假和谁审批了请假.
10. 用户表的用户ID对应党团员档案的用户ID外键,用于记录用户数据与党团员档案数据的对应关系.
11. 用户表的用户ID对应党团资料上传记录表的ID和录入人ID外键,用于记录谁上传了党团资料,谁审核了党团资料.
12. 用户表的用户ID对应推优任务表的发布人ID外键,用于记录谁创建了推优任务.
13. 用户表的用户ID对应推优报名表的用户ID外键,用于记录谁报名了推优.
14. 用户表的用户ID对应推优投票记录的投票人ID和被投票人ID外键,用于记录谁投票和投给了谁.
15. 用户表的用户ID对应活动表的发布人ID外键,用于记录谁发布了活动.
16. 用户表的用户ID对应活动报名记录表的用户ID外键,用于记录谁报名了活动.
17. 用户表的用户ID对应活动签到记录的用户ID外键,用于记录谁签到了.

# 1 功能 :
## 1 注册 :
学号+密码+再次输入密码
## 2 登录 :
学号+密码
## 3 个人信息管理
对应用户表, 成员变更记录表
用户对个人信息进行修改, 班长进行审核. 通过后保存到用户表中.每次变更记录在成员变更记录表中
## 4 通知管理
对应通知表, 通知已读记录表
1. 班长或团支书发送通知(包括标题, 内容, ,截止时间). 对应班级的成员接收通知 (包括通知的信息以及发布时间和发布人)
2. 检测成员是否已读通知, 已读的记录再通知已读记录中. 并对未读成员进行提醒

## 5 活动管理
对应活动表, 活动报名记录表, 活动签到记录表
## 6 考勤管理
对应请假申请表, 考勤任务表, 考勤记录表.
1. 学生可以提交请假申请.由团支书审核并返还审核意见及结果.
2. 团支书可以进行考勤数据录入 (谁没有来)
## 7 党团员管理
对应党团员档案表, 党团资料上传记录表.
1. 学生可以上传党团员资料, 经团支书审核后返回给学生审核结果, 并将通过的资料在党团员信息库中更新
2. 团支书可以直接修改党团员信息资料.
## 8 推优评议
对应推优任务表, 推优报名表, 推优投票记录表.
1. 团支书配置推优资格并发布推优任务并通知给指定班级的学生
2. 学生进行推优报名, 由团支书审核, 得到最终推优名单.
3. 团支书进行推优投票配置并发布推优投票.
4. 学生进行投票, 得到最终投票结果并进行公示.
## 9 公示与材料管理
对应公示表, 公示材料提交记录表.
1. 团支书配置公示并发布公示给指定班级的学生.
2. 如果有材料收集则学生提交材料及相关信息. 由团支书审核后返回给学生审核结果, 如果不通过需要重新提交. 团支书可以看到提交名单和材料.
