## 环境准备

### 1. 前置条件
- Python 3.12+
- MySQL 8.0+ 已安装并运行
- Anaconda (推荐)

### 2. 安装依赖

```bash
# 激活conda环境
conda activate djangoDev

# 进入项目目录
cd "D:\ProjectBase\SystemDesign\Project\党团班一体化系统"

# 安装依赖
pip install -r requirements.txt
```

### 3. 配置数据库

编辑项目根目录的 `.env` 文件，修改数据库连接信息：

```env
DB_NAME=党团班一体化系统
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_HOST=localhost
DB_PORT=3306
```

### 4. 创建数据库并迁移

```bash
# 数据库会自动创建，运行迁移：
python manage.py migrate

# 导入种子数据（50个用户）：
python manage.py seed_data
```

## 功能指南

### 1. 注册与登录

**注册:** 点击导航栏"注册" → 填写学号、姓名、性别、班级、手机号、密码 → 提交

**登录:** 输入学号和密码 → 点击"登录"

**退出:** 导航栏右上角用户名下拉 → "退出登录"

---

### 2. 个人信息管理

#### 学生视角
1. 点击导航栏用户名 → "个人信息" 查看个人资料和党团员档案
2. 点击"编辑信息"修改个人信息
3. **修改需要班长审核**，提交后等待审核
4. 可在个人信息页查看变更记录

#### 班长视角
1. 进入"个人信息" → 点击"审核成员变更"
2. 查看待审核的变更请求
3. 点击"通过"或"驳回"
4. 通过后修改立即生效

#### 团支书视角
1. 进入"个人信息" → 点击"党团员档案管理"
2. 查看全班党团员档案
3. 点击"编辑"直接修改档案信息

---

### 3. 通知管理

#### 发布通知（班长/团支书）
1. 导航栏 → "通知" → 点击"发布通知"
2. 填写标题、内容、截止时间
3. 点击"发布"

#### 接收通知（所有角色）
1. 导航栏 → "通知" 查看通知列表
2. 点击通知标题查看详情（自动标记为已读）

#### 追踪已读状态（班长/团支书）
1. 进入通知详情页 → 点击"查看已读状态"
2. 查看哪些学生已读/未读

---

### 4. 活动管理

#### 创建活动（班长/团支书）
1. 导航栏 → "活动" → "创建活动"
2. 填写活动名称、描述、时间、地点、签到码、签到截止时间
3. 点击"创建"

#### 报名活动（所有角色）
1. 在活动列表点击"详情"
2. 点击"报名"按钮
3. 可随时"取消报名"

#### 签到（所有角色）
1. 活动详情页 → 点击"签到"
2. 输入正确的签到码
3. 必须在签到截止时间前完成

#### 管理活动（班长/团支书）
1. 活动详情页 → 点击"管理"
2. 查看已报名名单和已签到名单

---

### 5. 考勤管理

#### 申请请假（学生）
1. 导航栏 → "考勤" → "申请请假"
2. 选择日期、填写原因
3. 提交后等待团支书审核

#### 查看请假（所有角色）
1. 导航栏 → "考勤" → "我的请假" 查看申请状态
2. 团支书可查看全班请假申请

#### 审核请假（团支书）
1. 考勤页 → "查看请假申请"
2. 找到待审核的申请 → 点击"审核"
3. 选择通过/驳回，填写意见

#### 创建考勤任务（团支书）
1. 考勤页 → "创建考勤任务"
2. 填写考勤名称、时间、地点

#### 录入考勤记录（团支书）
1. 在考勤任务中录入每个学生的考勤状态（出勤/缺勤/请假）
2. 可添加备注

---

### 6. 党团员管理

#### 上传资料（所有角色）
1. 导航栏 → "党团资料" → "上传资料"
2. 填写资料名称和存储路径
3. 提交后等待团支书审核

#### 审核资料（团支书）
1. 党团资料页 → 找到待审核记录 → 点击"审核"
2. 选择通过/驳回，填写审核意见

#### 查看资料库
1. 导航栏 → "党团资料" 查看所有上传记录及审核状态

---

### 7. 推优评议

#### 创建推优任务（团支书）
1. 导航栏 → "推优评议" → "发布推优任务"
2. 填写任务名称、报名截止时间、投票开始/结束时间
3. 点击"发布"

#### 报名推优（所有角色）
1. 在推优任务列表点击"详情"
2. 在报名阶段点击"报名"
3. 等待团支书审核

#### 审核报名（团支书）
1. 推优任务详情 → "审核报名"
2. 对每个报名者选择通过/驳回
3. 审核通过的学生成为候选人

#### 投票（所有角色）
1. 推优任务进入"投票中"阶段后
2. 进入任务详情 → "去投票"
3. 点击候选人旁的"投票"按钮
4. 每人只能投给同一候选人一次

#### 查看结果
1. 任务详情 → "查看结果"
2. 显示候选人得票排名

---

### 8. 公示与材料管理

#### 发布公示（团支书）
1. 导航栏 → "公示" → "发布公示"
2. 填写标题、内容
3. 设置是否需要提交材料及收集时间段
4. 点击"发布"

#### 查看公示（所有角色）
1. 公示列表查看所有公示
2. 点击查看详情

#### 提交材料（所有角色）
1. 公示详情页 → "提交材料"
2. 填写材料内容和附件路径
3. 如被驳回可重新提交

#### 审核材料（团支书）
1. 公示详情 → "审核材料"
2. 对每个提交选择通过/驳回，填写意见

## 常见操作流程

### 流程1: 成员信息变更（学生→班长）
```
学生编辑个人信息 → 生成变更申请 → 班长审核(通过/驳回) → 信息更新
```

### 流程2: 推优评议全流程（团支书→学生）
```
团支书发布推优任务 → 学生报名 → 团支书审核报名 → 
学生投票 → 查看投票结果
```

### 流程3: 活动签到（班长→学生）
```
班长创建活动+设置签到码 → 学生报名 → 学生输入签到码签到
```

### 流程4: 请假审批（学生→团支书）
```
学生提交请假申请 → 团支书审核(通过/驳回+意见) → 学生查看结果
```

### 流程5: 公示材料收集（团支书→学生）
```
团支书发布公示+开启材料收集 → 学生提交材料 → 
团支书审核(通过/驳回) → 驳回可重新提交
```

## 管理后台

Django 自带管理后台: **http://localhost:8000/admin/**

```bash
# 创建管理员账号
python manage.py createsuperuser
# 按提示输入学号、姓名、密码
```

管理后台可对所有数据表进行增删改查操作。
